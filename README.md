# Freelancer Document Templates

Diese Sammlung enthält professionelle Vorlagen für alle wichtigen Dokumente, die als IT-Freelancer benötigt werden.

## 📋 Verfügbare Templates

### Geschäftsdokumente
- **`angebot-template.adoc`** - Professionelles Angebot mit Kostenaufstellung
- **`rechnung-template.adoc`** - Rechnung mit allen steuerlichen Pflichtangaben
- **`agb-template.adoc`** - Allgemeine Geschäftsbedingungen für IT-Dienstleister

### Rechtliche Dokumente
- **`dienstvertrag-template.adoc`** - Freelancer-Dienstvertrag (Scheinselbständigkeit-sicher)
- **`nda-template.adoc`** - Geheimhaltungsvereinbarung (NDA)

### Projektmanagement
- **`statusbericht-template.adoc`** - Detaillierter Projekt-Statusbericht
- **`abnahmeprotokoll-template.adoc`** - Formelle Projektabnahme

## 🔧 Verwendung

### AsciiDoc Format
Alle Templates sind im AsciiDoc-Format (.adoc) erstellt, das folgende Vorteile bietet:

- **Professionelle Formatierung** - Automatische Inhaltsverzeichnisse, Tabellen, Admonitions
- **Export-Optionen** - PDF, HTML, Word-Export möglich
- **Versionskontrolle** - Git-freundliches Textformat
- **Wiederverwendbar** - Einfache Anpassung für verschiedene Projekte

### Template Anpassung

1. **Platzhalter ersetzen**: Alle `[GROSSBUCHSTABEN]` durch Ihre Daten ersetzen
2. **Firmenangaben**: Name, Adresse, Steuernummer eintragen
3. **Projektspezifisch**: Beschreibungen und Details anpassen
4. **Rechtliche Prüfung**: Templates von einem Anwalt prüfen lassen

### Export zu PDF

```bash
# AsciiDoctor installieren
gem install asciidoctor asciidoctor-pdf

# PDF generieren
asciidoctor-pdf template.adoc
```
